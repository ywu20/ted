# ted

## Prerequisites
- Make sure you have Python 3.7+ installed

## Launch code

- install the websockets library
```pip3 install websockets```

- setup the server  
```python3 s.py``` 

- open a new terminal and do flask (do we want flask or just open html pages)  
```python3 __init__.py```

- go to http://127.0.0.1:5000/
